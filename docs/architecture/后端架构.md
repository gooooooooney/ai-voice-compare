# 后端架构

## 服务架构

采用**纯前端架构** - 无后端服务器：

```
部署结构:
server/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── app.js              # 主应用逻辑
└── README.md           # 部署说明
```

## API密钥管理

```javascript
// 在app.js中直接配置
const CONFIG = {
  DEEPGRAM_API_KEY: 'your_deepgram_api_key_here',
  ASSEMBLYAI_API_KEY: 'your_assemblyai_api_key_here'
};
```
