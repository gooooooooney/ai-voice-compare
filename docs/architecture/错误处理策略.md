# 错误处理策略

## 统一错误处理

```typescript
class APIErrorHandler {
  static handleDeepgramError(error) {
    if (error.code === 4001) {
      return "Deepgram API密钥无效";
    }
    return "Deepgram连接错误";
  }
  
  static handleAssemblyAIError(error) {
    if (error.message.includes('token')) {
      return "AssemblyAI令牌无效或已过期";
    }
    return "AssemblyAI服务错误";
  }
}
```
